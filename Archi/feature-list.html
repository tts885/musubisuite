Frontend: React 19, TypeScript 5, Vite 6, TanStack Router, Axios, Tailwind CSS, shadcn/ui, Recharts, Sonner
Backend: Django 5.0, Django REST Framework, SimpleJWT, django-cors-headers, django-filter, drf-spectacular

📌 備考

開発環境:

  • DEBUG=True時は全てのCORSオリジンを許可
  • 一部APIエンドポイントは認証不要(開発環境のみ)
  • モックデータを使用している画面あり(将来的にAPI連携予定)

本番環境への移行時の注意:

  • CORS設定を特定オリジンのみに制限
  • 全APIエンドポイントで認証を必須化
  • SECRET_KEYの変更
  • DEBUG=False設定
  • PostgreSQLへのDB移行
  • 静的ファイル・メディアファイルの外部ストレージ化(Azure Blob Storage等)
